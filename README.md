# Sign Classifier
Это небольшая "нейронка", написанная на torch, которая может определять положительно число или нет. В SignClassifier1 лежит код на python, где расписана каждая строчка кода. Можно провести тест изменив test_number. Во втором файле без комментариев и с возможностью самому вводить цифры.
